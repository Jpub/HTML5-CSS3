sans-serif, serif 이외의 CSS 폰트 패밀리 키워드

CSS에서 폰트를 지정하려면 font-family 프로퍼티를 사용합니다. 쉼표로 연결해서 복수의 폰트를 지정해두면 첫 번째부터 순서대로 찾아서 적용합니다. 어떤 폰트로 존재하지 않는 경우를 대비해서 마지막에는 일반적인 폰트 패밀리를 지정해둡니다.

  • 5종류의 일반 폰트 패밀리 키워드
  • sans-serif
  • serif
  • cursive
  • fantasy
  • monospace

5류의 일반 폰트 패밀리 키워드

특정 폰트가 아닌 일반적인 폰트의 종류를 가리키는 키워드에는 sans-serif, serif, cursive, fantasy, monospace 등 5종류가 있습니다.

sans-serif

영문 폰트에선 '산세리프'를 가리키는 키워드입니다. 한글에선 '고딕체'에 해당합니다.

font-family: sans-serif;

serif

영문 폰트에선 '세리프'를 나타내는 키워드입니다. 한글에선 '명조체'에 해당합니다.

font-family: serif;

cursive

필기체 글자를 가리키는 키워드입니다. 한글에는 이에 해당하는 폰트가 없어서 명조체로 표시합니다.

font-family: cursive;

fantasy

장식 글자를 나타내는 키워드입니다. 한글에는 이에 해당하는 폰트가 없어서 고딕체 등으로 표시합니다.

font-family: fantasy;

monospace

san-serif, serif를 제외한 폰트 중에서 유일하게 자주 사용되는 것이 이 monospace입니다. 프로그램의 소스 코드 등을 표시할 때 자주 사용됩니다.

font-family: monospace;
